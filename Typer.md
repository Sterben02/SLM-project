**Что это:**  
**Typer** — библиотека Python для создания CLI (command-line interface) приложений, основанная на **Python type hints** (типах переменных). Это современная альтернатива `argparse` с лучшей поддержкой типов и автодополнения в редакторах.

**Простыми словами:**  
Typer помогает писать программы, которые запускаются из командной строки, но делает это проще и «умнее» чем старые библиотеки. Он использует типы переменных, поэтому:

- редактор автоматически подскажет параметры
- меньше ошибок при вводе
- код понятнее

**Пример с Typer:**

```python

import typer 
app = typer.Typer() 
@app.command() def greet(name: str, age: int = 18):     
	typer.echo(f"Привет, {name}! Возраст: {age}") 
	
if __name__ == "__main__":     
app()
```
Запуск:

```bash

python script.py --name Анна --age 25`
```

**Typer vs argparse:**

|Характеристика|Typer|argparse|
|---|---|---|
|Типы переменных|✅ использует type hints|❌ нужно вручную указывать|
|Автодополнение|✅ в редакторе|❌|
|Читаемость|✅ выше|❌ меньше|
|Сложность|проще|сложнее|
