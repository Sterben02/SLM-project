**Что это:**  
Фрагмент кода, содержащий уязвимость, ошибку или дефект, который может привести к нарушению безопасности: взлому программы, утечке данных, удалению файлов, выполнению команд злоумышленника.[](https://javarush.com/groups/posts/1585-samihy-opasnihy-kod)

**Простыми словами:**  
Это «дыра» в программе, через которую злоумышленник может проникнуть внутрь. Например:

- Если в коде есть SQL без проверки введённых данных → возможна SQL-инъекция
- Если пароль хранится в коде → любая утратка кода = утечка пароля
- Если программа выполняет команды пользователя без проверки → злоумышленник может запустить любой код

**Пример опасного кода:**

```python

# ❌ НЕБЕЗОПАСНО — SQL-инъекция 
query = f"SELECT * FROM users WHERE name = '{user_input}'"`
```

```python

# ✅ БЕЗОПАСНО — с параметрами 
query = "SELECT * FROM users WHERE name = ?" cursor.execute(query, (user_input,))
````

**Самые опасные коды:**

- **Fork-бомба** — программа, бесконечно создающая копии себя, пока не закончится память
```c

#include <unistd.h> 
int main() {     while(1) fork();  // создаёт бесконечные копии }`
```
