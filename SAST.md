**Что это:**  
SAST = **Static Application Security Testing** (статическое тестирование безопасности приложений) — анализ исходного кода ПО на наличие потенциальных уязвимостей без запуска программы.

**Простыми словами:**  
SAST — это «инспектор кода», который читает ваш код как книгу и говорит:

> «Вот здесь ты используешь SQL без проверки — сюда может внедриться злоумышленник!»  
> «Вот здесь ты хранит пароль в коде — это опасно!»

**Как работает:**

- Анализирует исходный код
- Поиск участков с потенциальными уязвимостями
- Не требуется запуск программы

**Плюсы SAST:**

- Раннее обнаружение уязвимостей (на этапе разработки)
- Полный охват кода
- Быстрее чем динамический тест (DAST)
