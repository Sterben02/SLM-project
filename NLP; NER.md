## **NLP** (Natural Language Processing)

**Что это:**  
NLP = **Natural Language Processing** (обработка естественного языка) — область машинного обучения, где компьютеры учатся понимать, интерпретировать и генерировать человеческий язык (текст и речь).

**Простыми словами:**  
NLP — это технология, которая делает возможным:

- умный перевод текста (Google Translate)
- чат-боты, которые понимают ваши вопросы
- автодополнение текста на смартфоне
- голосовые ассистенты (Siri, Яндекс.Станция)

**Что включает NLP:**
- работу с текстами на разных языках (русский, английский и др.)
- изучение структуры и правил языка
- создание систем, извлекающих смысл из текста и речи

## **NER** (Named Entity Recognition)

**Что это:**  
NER = **Named Entity Recognition** (распознавание именованных сущностей) — задача в области NLP, где модель выделяет и классифицирует значимые фрагменты текста: имена людей, организации, города, даты, суммы денег и т.п..

**Простыми словами:**  
NER — это «умный маркер», который проходит по тексту и говорит:

> «Это имя человека» → **PER** (персона)  
> «Это город» → **LOC** (местоположение)  
> «Это организация» → **ORG**  
> «Это дата» → **DATE**

**Пример:**

```
Текст: "Париж — столица Франции. Иван Петров работал в компании Google в 2020 году." NER выделяет: Париж → LOC (город) Франция → LOC (страна) Иван Петров → PER (персона) Google → ORG (организация) 2020 → DATE (дата)
```

**Как работает NER:**
- Использует машинное обучение и NLP
- Обнаруживает слово или строку слов (например, «Нью-Йорк»)
- Знает категорию объекта

**Сложности NER:**

| Проблема              | Пример                                       |
| --------------------- | -------------------------------------------- |
| Языковое разнообразие | «Париж» пишется разжно на разных языках      |
| Контекст              | «Вена» = город (LOC) или кровеносный сосуд?  |

**Где применяется NER:**
- Обогащение обучающих выборок признаками
- Анализ связей между сущностями (Named Entity Linking)
- Поиск, рекомендательные системы, анализ документов
