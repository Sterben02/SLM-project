
**Что это:**  
**Entropy-based detector** — детектор, который использует **энтропию** (метрику случайности) для поиска секретов в коде. Секреты (пароли, ключи) имеют высокую энтропию, а обычный текст — низкую.

**Простыми словами:**  
Энтропия — мера «случайности» данных:

- пароль `a1B2c3D4e5F6g7H8` → высокая энтропия (случайный)
- слово `привет` → низкая энтропия (обычный текст)

Детектор проверяет: если строка в коде очень «случайная» → возможно, это секрет.

**Пример:**

```python
import math 
from collections import Counter 

def entropy(s):     
	"""Вычисляет энтропию строки"""    
	counter = Counter(s)    length = len(s)    
	return -sum((count/length) * math.log2(count/length) for count in counter.values()) 
	
test_strings = [     
	"password123",      # низкая энтропия    
	"a1B2c3D4e5F6g7H8",  # высокая энтропия 
	] 

for s in test_strings:     
	print(f"{s}: {entropy(s)}")
```

**Плюсы:**

- не зависит от паттернов
- хорошо ловит случайные ключи

**Минусы:**

- может давать ложные совпадения (например, хеш)
- требует настройки (threshold)
